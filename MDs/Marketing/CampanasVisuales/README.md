Esta carpeta define **qué imágenes debe crear Andre manualmente** para las campañas de Infinity Ascend usando un generador de mejor calidad visual —por ejemplo GPT Image, Midjourney, Leonardo, DALL·E, Sora image, Photoshop/Firefly u otra herramienta externa— sin perder coherencia con el universo visual ya existente en el repo.

La regla es simple:

```text
Primero referencias del repo.
Después prompt.
Después generación.
Después selección.
Después subida al repo.
```

No generar imágenes genéricas de “sci-fi helmet” sin mirar el material existente.

## Regla visual de pertenencia

Una imagen nueva solo pertenece a Infinity Ascend si contiene al menos **3 de estos elementos**:

- figura humana solitaria;
- casco negro brillante con interior cósmico azul;
- luz vertical / portal / beam central;
- paleta azul, cian, violeta y dorado;
- amanecer o horizonte luminoso;
- reflejos en agua, hielo o suelo pulido;
- escala monumental;
- atmósfera espiritual sci-fi;
- sensación de paz, no agresión;
- viaje interior / ascensión.

## Lo que NO se debe hacer

Evitar:

- casco genérico de astronauta sin relación con las referencias;
- estética EDM festival, rave, neón agresivo o DJ stage;
- exceso de cyberpunk sucio si no hay paz/trascendencia;
- personajes anime/cartoon si no se pide explícitamente;
- texto dentro de la imagen generado por IA;
- logos inventados;
- tipografías deformes;
- rostros humanos hiperrealistas innecesarios;
- demasiados elementos pequeños que no se entiendan en móvil.

## Configuración recomendada por formato

### Instagram Feed

- Aspect ratio: **1:1**.
- Resolución ideal: 2048x2048 o superior.
- Uso: identidad, lore, carrusel, posts guardables.
- Composición: limpia, un centro visual fuerte.

### Reels / TikTok / Shorts

- Aspect ratio: **9:16**.
- Resolución ideal: 1080x1920 o 2160x3840.
- Uso: video vertical con zoom/parallax.
- Composición: sujeto o luz en el centro, dejando espacio superior para texto.

### YouTube thumbnail

- Aspect ratio: **16:9**.
- Resolución ideal: 1920x1080.
- Uso: visualizer, lyric video, videoclip.
- Composición: alto contraste, silueta clara, no saturar.

### Spotify Canvas

- Aspect ratio: **9:16**.
- Duración final: 3–8 segundos si se anima.
- Uso: loop visual sin texto.
- Composición: movimiento simple, luz, portal, casco o horizonte.

### Landing / web hero

- Aspect ratio: **16:9** o panorámico.
- Resolución ideal: 2560x1440 o superior.
- Uso: portada web, banner, cabecera.
- Composición: mucho espacio negativo para texto HTML encima.

# Campaña 1 — AION / The Helm Singularity

## Objetivo

Presentar a **AION** como la singularidad integrada dentro del casco de Infinity Ascend.

Esta campaña no debe verse como “robot asistente genérico”. Debe verse como:

```text
el casco contiene una consciencia / un universo / una guía interna
```

## Uso

- Post de lore en Instagram.
- Avatar alternativo.
- Carrusel explicando AION.
- Intro de una serie: “AION Transmission 001”.
- Story destacada: `AION` o `Lore`.

## Referencias principales

- `Imagenes/contextuales/Thehelmet.png`
- `Imagenes/contextuales/InfinityHelmet.png`
- `Imagenes/contextuales/InfinityHelmet2.png`
- `Imagenes/Videoclips/InfinityAscend/7.png`
- `Imagenes/Videoclips/InfinityAscend/13.png`

## Imágenes a crear

### 1A — Retrato del casco singularidad

- Formato: 1:1.
- Imagen: casco protagonista flotando o sobre pedestal.
- Debe verse el interior cósmico azul dentro del visor.
- Fondo oscuro, limpio, con aura cian/dorada.
- Sin texto.

Prompt sugerido:

```text
Use the provided Infinity Ascend helmet references as the main design source. Create a square premium Instagram artwork of the same glossy black futuristic helmet, rounded shell, oversized dark visor containing a deep blue starfield and a quiet glowing singularity inside. Add subtle cyan light strips and a spiritual sci-fi aura. The helmet should feel like a sacred technological artifact, not a generic astronaut helmet. Dark blue cosmic background, soft golden rim light, calm, elegant, mysterious, high-end album cover quality, no text, no logo, no watermark.
```

### 1B — AION despertando dentro del casco

- Formato: 9:16.
- Imagen: casco en laboratorio blanco, luz azul saliendo desde dentro.
- Inspiración: activación / despertar / singularidad online.
- Debe servir para Reel/TikTok.

Prompt sugerido:

```text
Using the Infinity Ascend helmet reference images, create a vertical 9:16 cinematic image of the same glossy black helmet inside a clean white futuristic laboratory. A blue-white singularity awakens inside the visor, casting soft cyan light onto the room. The scene should feel calm, sacred and technological, like the first activation of AION. Minimal composition, premium sci-fi, spiritual atmosphere, no aggressive cyberpunk, no text, no logos.
```

### 1C — AION transmisión

- Formato: 1:1 o 9:16.
- Imagen: casco frente a una interfaz abstracta de ondas/resonancia.
- Útil para posts de “AION dice / AION analiza / AION transmission”.

Prompt sugerido:

```text

The helmet contains a starfield singularity in the visor. Background should look like a classified spiritual-science interface, elegant and minimal. Color palette: deep blue, cyan, black, silver, warm white. No readable text, no fake letters, no logo.
```

## Hooks recomendados

```text
AION has awakened.
The helmet contains a universe.
A signal from inside the helmet.
The singularity is listening.
```

## Caption recomendado

```text

A quiet intelligence inside the Infinity Ascend helmet.
Music for inner ascension.
```

# Flujo manual recomendado para Andre

## Paso 1 — Elegir campaña

Elegir una de estas:

```text
AION / Peace / Portal / Dreams / Helmet Lore / Spotify Release / YouTube Visualizer / Landing
```

## Paso 2 — Abrir referencias

Abrir las imágenes indicadas en cada campaña. Si el generador permite subir imágenes, subir de 2 a 5 referencias.

## Paso 3 — Copiar prompt

Copiar el prompt sugerido y ajustarlo solo si hace falta.

## Paso 4 — Generar 4 variantes mínimo

No aceptar la primera. Generar al menos:

```text
v1, v2, v3, v4
```

## Paso 5 — Elegir con checklist

La imagen debe pasar estas preguntas:

- ¿Se parece al universo visual existente?
- ¿El casco respeta la silueta y materiales del repo?
- ¿Funciona en móvil?
- ¿Tiene paz y ascensión, no agresión?
- ¿Se entiende en 1 segundo?
- ¿No contiene texto deformado?
- ¿Sirve para el canal elegido?

## Paso 6 — Guardar con nombre ordenado

Usar nombres como:

```text
Imagenes/Campanas/AION/aion_helm_singularity_01.png
Imagenes/Campanas/Peace/peace_inner_ascension_01.png
```
