# Roles operativos necesarios — Infinity Ascend

## Propósito

Este documento define cuántos trabajos o funciones distintas necesita Infinity Ascend para operar bien como proyecto musical, visual y de marketing.

No significa que hagan falta 22 personas. Significa que el proyecto contiene **22 funciones** que normalmente existirían dentro de un equipo musical, creativo o de marketing.

La visión futura es que Andre pueda apoyarse en **AION + cronjobs + repo** para automatizar o asistir una parte importante de estas funciones, de modo que idealmente Andre tenga que concentrarse cada vez más en:

```text
crear música + aprobar decisiones importantes + publicar cuando haga falta
```

La victoria absoluta sería:

```text
AION investiga → propone → organiza → genera prompts → crea calendarios → analiza métricas → recomienda decisiones
Andre crea/publica música → aprueba lo importante → mantiene visión artística
```

---

# Conteo general

Para hacer bien Infinity Ascend se identifican aproximadamente:

```text
22 funciones operativas distintas
```

Estas funciones se dividen en cuatro grandes áreas:

1. Núcleo creativo.
2. Contenido y redes.
3. Distribución y crecimiento.
4. Sistema operativo y automatización.

---

# 1. Director creativo

Define el universo general:

- qué es Infinity Ascend;
- qué estética tiene;
- qué emociones transmite;
- qué no debe hacer;
- qué símbolos son sagrados: casco, portal, luz, ascensión, AION.

Este rol es clave porque sin dirección creativa todo se vuelve contenido random.

## Automatización posible

AION puede asistir mucho, pero no reemplazar completamente. La decisión final debe seguir siendo de Andre.

---

# 2. Brand strategist / estratega de marca

Define:

- posicionamiento;
- one-liner;
- promesa emocional;
- diferenciación;
- target;
- tono de comunicación.

Ejemplo:

```text
Soft futuristic electropop for dreamers seeking peace, light, and transcendence.
```

## Automatización posible

Alta. AION puede mantener documentos de marca, revisar consistencia y proponer mejoras.

---

# 3. Lore / worldbuilding writer

Construye la mitología:

- AION;
- el casco;
- la singularidad;
- el portal;
- la ascensión;
- transmisiones;
- símbolos;
- narrativa del universo.

Este rol es importante porque Infinity Ascend no es solo música: es un mundo.

## Automatización posible

Media-alta. AION puede generar lore, transmisiones y documentos, pero Andre debe aprobar el canon.

---

# 4. Copywriter

Escribe:

- captions;
- hooks;
- bios;
- frases para posts;
- textos para landing;
- CTAs;
- guiones cortos para Reels/TikToks.

Ejemplos:

```text
For people who need peace, not noise.
The helmet contains a universe.
A soft portal for your mind.
```

## Automatización posible

Muy alta. Es uno de los roles más automatizables con cronjobs de texto.

---

# 5. Community manager

Gestiona:

- calendario de publicación;
- respuestas a comentarios;
- DMs;
- stories;
- interacción con seguidores;
- registro de feedback emocional.

Este rol conecta la marca con personas reales.

## Automatización posible

Media. AION puede preparar respuestas, detectar temas y resumir feedback. La interacción humana auténtica conviene que siga pasando por Andre al principio.

---

# 6. Social media manager

Diseña el sistema de redes:

- qué va a TikTok;
- qué va a Instagram;
- qué va a YouTube Shorts;
- qué va a YouTube largo;
- qué se recicla;
- qué se publica cada día.

Es distinto al community manager: uno diseña la estrategia de redes, el otro gestiona la comunidad.

## Automatización posible

Alta. AION puede producir calendarios, prioridades y planes semanales.

---

# 7. Content strategist

Decide qué contenidos testear:

- peace;
- portal;
- dreams;
- helmet/lore;
- lyric quotes;
- Spotify conversion.

Este rol mira qué contenido tiene sentido para validar audiencia.

## Automatización posible

Muy alta. Se puede convertir en cronjobs de ideas, tests y matrices de decisión.

---

# 8. Short-form video editor

Produce:

- TikToks;
- Reels;
- YouTube Shorts;
- loops verticales;
- cortes de visualizers;
- clips con zoom, parallax, texto y música.

Para Infinity Ascend, este rol será muy importante.

## Automatización posible

Media-baja si hablamos solo de texto. Alta si en el futuro se añaden herramientas de video/plantillas, pero no entra en el scope de cronjobs solo texto.

---

# 9. Motion designer

Anima imágenes:

- movimiento lento;
- partículas;
- beams de luz;
- portal abriéndose;
- casco activándose;
- loops para Spotify Canvas;
- visualizers.

Diferente al editor: el motion designer crea movimiento gráfico/visual.

## Automatización posible

Baja con cronjobs solo texto. Puede asistir con instrucciones, prompts y shotlists, pero no producir motion final sin herramientas visuales.

---

# 10. Graphic designer

Crea:

- portadas;
- posts;
- carouseles;
- thumbnails;
- banners;
- layouts;
- overlays;
- assets de campaña.

Para este proyecto debe trabajar siempre con referencias del repo.

## Automatización posible

Baja-media con cronjobs solo texto. AION puede crear briefs, prompts, checklists y criterios, pero no reemplaza la ejecución visual final si no usamos generación/edición de imagen.

---

# 11. AI image art director

No solo genera imágenes. Dirige IA visual:

- selecciona referencias;
- escribe prompts;
- evalúa outputs;
- descarta imágenes malas;
- mantiene coherencia visual;
- documenta prompts ganadores.

Este rol será fundamental si se usan GPT Image, Midjourney, Leonardo u otras herramientas.

## Automatización posible

Media con cronjobs de texto. Puede preparar prompts y briefs; no puede aprobar calidad visual sin ver imágenes.

---

# 12. Music producer

Hace o mejora la música:

- estructura;
- sonido;
- arreglos;
- mezcla emocional;
- identidad sonora;
- versión corta para redes;
- versión completa para Spotify.

Andre conserva mucho control creativo aquí.

## Automatización posible

Baja con cronjobs solo texto. AION puede sugerir estructura, letras, referencias y decisiones, pero no reemplaza producción musical final.

---

# 13. Mixing / mastering engineer

Prepara la música para publicación profesional:

- mezcla;
- mastering;
- loudness;
- claridad;
- compatibilidad Spotify/YouTube/TikTok;
- exportaciones correctas.

Aunque la música sea buena, si suena amateur baja la conversión.

## Automatización posible

Baja con cronjobs solo texto. Puede crear checklist técnico y analizar notas, pero no masterizar audio desde texto.

---

# 14. Release manager

Gestiona lanzamientos:

- distributor;
- fechas;
- metadata;
- portada;
- Spotify for Artists;
- pitch a playlists;
- links;
- pre-save si aplica.

Este rol convierte canciones en releases reales.

## Automatización posible

Alta como sistema de checklist, recordatorios y preparación de metadata. La publicación final suele requerir acción manual.

---

# 15. Spotify / streaming strategist

Optimiza:

- perfil de artista;
- bio;
- canvas;
- playlists;
- saves;
- followers;
- track sequencing;
- métricas de streaming.

No es lo mismo que subir una canción. Es trabajar para que la canción convierta.

## Automatización posible

Media-alta. AION puede preparar bios, pitches, análisis y reportes si Andre introduce métricas o hay APIs disponibles.

---

# 16. YouTube strategist

Gestiona:

- canal;
- visualizers;
- lyric videos;
- Shorts;
- thumbnails;
- títulos;
- descripciones;
- SEO;
- playlists.

YouTube puede ser archivo audiovisual + descubrimiento.

## Automatización posible

Alta en texto: títulos, descripciones, tags, guiones, estructura de publicación y análisis manual de métricas.

---

# 17. TikTok / Reels trend researcher

Investiga:

- trends de música;
- formatos virales;
- hooks similares;
- sonidos;
- captions;
- patrones de retención;
- qué están haciendo artistas parecidos.

Este rol es buen candidato para cronjobs futuros.

## Automatización posible

Alta si se basa en investigación web/texto. Media si requiere ver muchos videos directamente.

---

# 18. Data analyst / growth analyst

Mide:

- views;
- retención;
- shares;
- saves;
- comments;
- follows;
- clicks;
- Spotify saves;
- conversiones.

Luego decide:

```text
repetir / modificar / matar / escalar
```

Sin este rol, se trabaja por intuición.

## Automatización posible

Muy alta si Andre registra métricas o se integran fuentes de datos.

---

# 19. Project manager

Ordena el sistema:

- tareas;
- prioridades;
- qué va primero;
- qué falta;
- qué está bloqueado;
- qué publicar esta semana;
- qué revisar el domingo.

En este proyecto, esto puede vivir muy bien en repo + cronjobs + AION.

## Automatización posible

Muy alta. Es uno de los mejores candidatos para cronjobs.

---

# 20. Web / landing designer

Crea:

- link-in-bio;
- landing;
- página del proyecto;
- CTA a Spotify;
- sección de lore;
- visual hero;
- rutas de conversión.

No necesita ser una web enorme al principio, pero sí clara.

## Automatización posible

Media con texto. AION puede crear copy, estructura y specs. La ejecución visual/web completa depende de herramientas de diseño/desarrollo.

---

# 21. Automation engineer

Diseña cronjobs y pipelines:

- generación de ideas;
- calendario semanal;
- análisis de métricas;
- investigación de trends;
- prompts visuales;
- resúmenes de performance;
- reportes diarios/semanales.

Este rol es el puente hacia la visión de dejar trabajar al sistema.

## Automatización posible

Muy alta. AION puede diseñar y mantener gran parte de este sistema.

---

# 22. Artist / performer / creator

Este rol sigue siendo Andre.

Aunque todo se automatice, Andre sigue siendo:

- fuente de la música;
- criterio final;
- sensibilidad;
- decisión artística;
- cara/creador detrás del universo;
- quien publica o aprueba.

La automatización no debe reemplazar el criterio artístico. Debe protegerlo.

## Automatización posible

Baja. Se puede asistir, pero no sustituir.

- social media planner;
- campaign planner;
- trend researcher;

- project manager;
- automation engineer;

- criterio artístico final;

# Conclusión

Infinity Ascend no es solo un proyecto musical. Es una operación creativa completa que mezcla:

```text
música + visuales + lore + marketing + datos + automatización
```

Por eso el conteo real no es “hacer canciones y subir posts”. El conteo real es coordinar unas:

```text
22 funciones operativas
```

La versión lean para Andre + AION no necesita contratar 22 personas. Necesita construir un sistema donde:

```text
Andre mantiene visión artística y música.
AION organiza, escribe, analiza, propone y automatiza.
Cronjobs ejecutan rutinas recurrentes.
El repo conserva memoria, estructura y decisiones.
```

Ese es el camino hacia la mini-agencia interna de Infinity Ascend.
