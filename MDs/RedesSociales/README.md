# Redes Sociales — Infinity Ascend

Esta carpeta contiene el sistema de creación y operación de canales sociales para Infinity Ascend.

## Orden de lectura

1. `PLAN_CREACION_CANALES.md`  
   Plan maestro para crear el ecosistema completo desde cero.

2. `REGISTRO_CANALES.md`  
   Archivo vivo para registrar URLs, handles y estado real de cada canal.

3. `PRIORIDADES_TARGET.md`  
   Orden de prioridad por target: idioma, países, redes, frecuencia y métricas.

4. `../Marketing/CALENDARIO_30_DIAS.md`  
   Calendario general de publicación y validación para TikTok, Reels, Shorts, Instagram, YouTube y Spotify/landing.

5. Canal por canal:
   - `Spotify.md`
   - `Tiktok.md`
   - `Instagram.md`
   - `Youtube.md`
   - `Facebook.md`
   - `LANDING_LINK_IN_BIO.md`

6. Carpeta específica de Instagram:
   - `Instagram/PLAN_INSTAGRAM.md`
   - `Instagram/CALENDARIO_30_DIAS_INSTAGRAM.md`
   - `Instagram/PROMPTS_INSTAGRAM.md`

## Principio central

Infinity Ascend no debe comunicarse como EDM genérico.

La posición estratégica es:

> Soft futuristic electropop for dreamers seeking peace, light, and transcendence.

Mensaje corto:

> Music for inner ascension.

## Sistema de canales

```text
TikTok / Shorts / Reels
        ↓
Instagram / YouTube
        ↓
Spotify / Landing
        ↓
seguimiento, guardados, comunidad
```

## Prioridades

### Primero

- Spotify o destino de escucha.
- TikTok para descubrimiento.
- Instagram para identidad visual.
- YouTube para archivo audiovisual.

### Después

- Facebook como presencia básica.
- Landing/link-in-bio como centro de enlaces.

## Uso práctico

Cuando se vaya a crear un canal:

1. Abrir su archivo específico.
2. Copiar bio, assets y checklist.
3. Crear cuenta.
4. Registrar URL/handle en `REGISTRO_CANALES.md`.
5. Publicar los primeros posts sugeridos.
6. Medir y actualizar aprendizajes.
